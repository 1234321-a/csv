## Contribution

We works under test driven development.

Checkout master source code from github:

```
hub clone goodby/csv
cd csv/
```

Install phpunit and mockery via composer:

```
./scripts/bundle-devtools.sh .
```

Run phpunit:

```
./vendor/bin/phpunit
```
